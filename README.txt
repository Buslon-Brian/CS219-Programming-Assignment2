# Project part 2 For Cs219 Computer Organization
>This program reads from an input file and translates the text from assembly language to execute the commands 
and display the results


[![WSL: Ubuntu-20.04][wsl-image]][https://docs.microsoft.com/en-

> Install Linux or the windows subsystem

> Navigate to build folder
> enter make into terminal 
> enter ./PROJECT2 into terminal
